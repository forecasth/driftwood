# Driftwood

Driftwood is a front-end-only, atmospheric space built with React, Vite, and Three.js.
It is not a game or a tool.
It is a quiet environment that breathes, drifts, and waits.

## Run

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## Philosophy

Driftwood is designed as restraint:
no accounts, no scores, no dashboards, no instructions on screen.

Enter the scene like dusk in a forest:
still air, distant movement, and the sense that something is alive just beyond sight.
